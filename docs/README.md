## 3. 核心功能亮點
*   **AI 語義診斷**: 串接本地 Qwen-2.5-Coder，自動分析 SQL 錯誤並提供繁體中文修正建議。
*   **4-Tier Metadata**: 支援 `DB -> Schema -> Table -> Column` 的完整語義校驗。
*   **專業 TUI**: 基於 JLine 的互動式 Shell，具備現代化 ANSI 表格渲染。
*   **安全防護**: 強制唯讀保護 (SELECT-only)，阻斷 DDL/DML 安全威脅。
